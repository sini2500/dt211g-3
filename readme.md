## DT211G - frontend-baserad webbutveckling

Det här är ett projekt för kursen frontend-baserad webbutveckling, VT 2026.

Webbplatsen är publicerad på Netlify: https://dt211g-3-sini2500.netlify.app/

---

Lab 3 går ut på att använda SCSS för att ge stil åt en webbplats.

Jag har valt att göra en enkel rankingsajt där användare kan rösta på bäst katt.

Det finns bara några få bilder att rösta på, och resultat sparas bara i localstorage.
